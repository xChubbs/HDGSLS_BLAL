# Un método de Galerkin Discontinuo Hibridizable para la ecuación de Schrödinger estacionaria lineal
Rescurso electronico asociado al trabajo de grado.

## Resumen
El artículo desarrolla un análisis numérico completo del método de Galerkin Discontinuo Hibridizable
(HDG) aplicado a la ecuación de Schrödinger estacionaria lineal, abordando tanto el problema fuente como el
problema de autovalores asociado al operador Hamiltoniano. Se presentan los fundamentos teóricos necesarios,
incluyendo espectro de operadores, resolvente, convergencia y formulación variacional, posteriormente se
construye el esquema HDG, definiendo espacios discretos, proyecciones y condiciones de acoplamiento del
esqueleto al mallado. Se establece la existencia, unicidad y propiedades espectrales del operador discreto,
demostrando cotas de error para las variables volumétricas y de frontera, y mostrando cómo estas permiten
recuperar tasas de convergencia óptimas y superconvergencia para autovalores y autofunciones. Finalmente,
se validan los resultados mediante ejemplos numéricos representativos, como la caja de potencial y el oscilador
armónico cuántico como ejemplos clásicos de la mecánica cuántica y posteriormente se evalua el método
aplicado al caso de la molecula de agua.
